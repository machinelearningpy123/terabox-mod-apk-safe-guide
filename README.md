# TeraBox APK Guide

Static GitHub Pages site created for the drag-and-drop deployer bot.

## Deploy

Drag this ZIP onto `DRAG_SITE_ZIP_HERE.bat` from the GitHub Pages deployer package.

## Includes

- index.html
- pages/safe-download.html
- pages/alternatives.html
- pages/faq.html
- 404.html
- assets/style.css
- assets/script.js
- robots.txt
- sitemap.xml
- .nojekyll
- site_config.json

## Note

This site is an independent informational guide. It does not host cracked or modified APK files.
